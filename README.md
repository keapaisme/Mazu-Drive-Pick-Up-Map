# Baishatun Mazu Pilgrimage Car Support Map  
# 白沙屯媽祖進香開車接駕規劃地圖

**I’m joining the Baishatun Mazu Pilgrimage this year, but I’ll be driving my own car instead of walking.**  
This project is a practical map tool built for car-based pilgrimage planning. It helps identify possible overnight stop areas, historical reference points, parking observation spots, and nearby National Freeway No. 1 / No. 3 interchanges along the route to Beigang. It is designed as a planning aid for route observation, pickup timing, and flexible driving support during the pilgrimage.  

這個專案是一個以 **開車接駕／跟駕規劃** 為核心的實用地圖工具，用來整理白沙屯媽祖前往北港進香路線上的 **候選駐駕區、歷年參考點、停車觀察點**，以及鄰近的 **國道一號／國道三號交流道**。  
它的用途是協助使用者進行 **路線觀察、接駕時機判斷、夜間等候與彈性開車支援規劃**。  

> This is **not an official route map**.  
> All locations and suggestions are for reference and tactical planning only.  
> Actual movement should always follow the live pilgrimage route, traffic control, and on-site conditions.

> 本圖 **不是官方路線圖**。  
> 所有點位與建議皆屬於 **規劃示意與戰術參考用途**，  
> 實際行程仍應以 **即時路線、交管措施與現場狀況** 為準。
